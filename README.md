# Prototype Template Workflow

This repo is a reusable starting point for fast product prototypes.

## Usage

1. Copy this repo using "Use this template"
2. Immediately write `SPEC.md`
3. Implement only the core flow
4. Deploy to Vercel and share preview link

### Rules
- No auth unless essential
- No backend unless necessary
- No database unless unavoidable
- Optimize for clarity, not architecture

### Definition of Done
- A user can experience the core idea in <30 seconds
- A shareable URL exists
- The concept can be evaluated by a human without explanation
